# Practice - Discord.js

## Commands

### Setup

```shell
git clone https://...
npm ci
cp config.json.sample config.json

# Input your bot's token to config.json
```

### Run

```shell
node .
```

### Registering slash commands

```shell
node deploy-commands.js
```
